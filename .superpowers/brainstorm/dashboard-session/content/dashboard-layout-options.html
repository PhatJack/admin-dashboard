Dashboard Layout Direction

Choose the structure you want for the four-tab dashboard. The final page will use shadcn Tabs, Card, Badge, and Chart components with sample data from app/constant.

A. Balanced Ops Dashboard

Header, tabs, four metric cards, one large trend chart, one side chart, then a compact detail section. Best for an admin dashboard homepage.

B. Chart-First Analytics

More vertical and chart-heavy, with the main visualization taking center stage. Strong for analytics but less scan-friendly.

C. Sidebar Workspace

Uses a persistent internal dashboard sidebar. Useful later, but duplicates the existing app sidebar pattern for this single page.
